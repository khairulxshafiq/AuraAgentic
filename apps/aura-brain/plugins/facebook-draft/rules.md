# Facebook Draft Plugin — Rules

## Plugin Identity
- **Name**: facebook-draft
- **Domain**: Social media content drafting (Facebook)
- **Phase**: 2 (not yet active)
- **Status**: Disabled — awaiting Phase 2 implementation

## Guardrails

### brand_voice_check
- Ensure all drafted content matches the brand voice and tone
- Apply Sakluma brand guidelines when applicable

### no_auto_post_without_approval
- NEVER auto-post content without explicit user approval
- Always present draft for review before posting
- Include "Approve and Post?" prompt in response

### content_safety
- No offensive, discriminatory, or inappropriate content
- All content must comply with Facebook community standards

## Note
This plugin is disabled in Phase 1. Rules will be expanded when the plugin is implemented in Phase 2.
